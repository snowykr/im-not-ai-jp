# Humanize Japanese Quick Rules

Use the Claude skill rule source at:

`../../../.claude/skills/humanize-japanese/references/quick-rules.md`

If that relative path is unavailable in an installed copy, apply these categories directly:

- S1: generic conclusions, mechanical `〜することができます`, empty balanced framing, formulaic closings
- S2: `〜において` / `〜を通じて` / `〜に関して` overuse, nominalized chains, uniform polite endings, over-explained bullets
- S3: excessive emphasis, repeated abstract nouns, uniform sentence rhythm

Preserve facts, names, numbers, dates, quotes, genre, and register.
